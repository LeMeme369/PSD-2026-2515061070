# SISTEM TO-DO LIST (1D LIST)
## a. Judul program
sistem To-do list menggunakan 1D list
## b. deskripsi singkat
Program ini dibuat sebagai aplikasi sederhana untuk mengelola daftar tugas (to-do list) melalui terminal. Dari sisi struktur data dan algoritma, program ini menggunakan list 1 dimensi (dynamic array) sebagai tempat penyimpanan data. Operasi yang diterapkan meliputi penambahan elemen (append), penghapusan elemen (pop), iterasi menggunakan perulangan (for dengan enumerate), serta manipulasi data berbasis indeks.

Penjelasan Kode 
1. Fungsi menu
- untuk menunjukan menu kepada user
2. Fungsi main
- menggunakan pengulangan while true
- menjalankan fungsi sesuai dengan pilihan user
3. fungsi append
- untuk menginput nilai yang di berikan kedalam list
4. fungsi enumerate
- untuk mengulang melalui sebuah iterable—seperti list, tuple, atau string—dan memiliki akses ke indeks dan elemen itu sendiri
- untuk memudahkan menampilkan isi list
5. fungsi pop
- untuk mengeluarkan data yang ada didalam list tersebut
